How to Use This Toolkit
  • Fill in the Emergency Info sheet first — it's the most important
  • Complete Medical Records with current diagnoses and history
  • Add all medications to the Medications sheet
  • List all doctors and specialists in Providers
  • Record key legal documents in Legal & Finance
  • Use Appointments to track upcoming visits
  • Use Family Notes for coordination between caregivers
  • Print any sheet using the Print button above
  • Save all sheets as PDF using Save All button
